deps-package

Custom Linux Package Repository

Setup Instructions

1. Register the repository GPG key (Recommended):

# Unsigned repository (less secure)

2. Add the repository to sources list:

echo "deb [trusted=yes] https://octaoss.github.io/deps-package/dependency/debian stable main" | sudo tee /etc/apt/sources.list.d/deps-deps-package.list

3. Update package index and install packages:

sudo apt update
sudo apt install <package-name>

1. Add the repository configuration file:

sudo curl -sL https://octaoss.github.io/deps-package/dependency/fedora/deps-deps-package.repo -o /etc/yum.repos.d/deps-deps-package.repo

2. Update metadata cache and install packages:

sudo dnf makecache
sudo dnf install <package-name>
